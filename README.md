Hikar Webapp
============

This is the repository for the [AR.js](https://github.com/AR-js-org/AR.js) based Hikar web app, which aims to port the [Hikar](https://hikar.org) Android app to the web. 

### IMPORTANT!

This is the current production version, available at [https://hikar.org/webapp](https://hikar.org/webapp). It is based on "classic" location-based AR.js and A-Frame, but does not support iOS and will not see any further development. 

New versions will switch to LocAR.js/RDK. Initial experiments available 
[here](https://github.com/nickw1/hikar.js-rdk)
