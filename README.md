Hikar Webapp
============

This is the repository for the [AR.js](https://github.com/AR-js-org/AR.js) based Hikar web app, which aims to port the [Hikar](https://hikar.org) Android app to the web. 

### IMPORTANT!

This is the current production version, available at [https://hikar.org/webapp](https://hikar.org/webapp). However it will not see any further development. Development is switching to [a new version using LocAR.js and RDK](https://github.com/nickw1/hikar.js-rdk)
